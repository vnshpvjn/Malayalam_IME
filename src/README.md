# Enhanced Malayalam IME - Source Files

This folder contains all source code files for the enhanced Malayalam Input Method Editor.

## Files

- **MalayalamIME.cpp** - Main IME implementation with 150+ enhanced chillu character mappings
- **MalayalamIME.h** - Main header file with class definitions
- **Register.cpp** - COM registration implementation
- **stdafx.h** - Precompiled headers
- **stdafx.cpp** - Precompiled source file
- **targetver.h** - Windows version targeting
- **MalayalamIME.def** - Module definition file
- **MalayalamIME.rc** - Resource file
- **MalayalamIME.vcxproj** - Visual Studio project file
- **MalayalamIME.sln** - Visual Studio solution file

## Building

Use the GitHub Actions workflow (`.github/workflows/build.yml`) for automatic cloud compilation, or follow the local build instructions in `../docs/BUILD_INSTRUCTIONS.md`.
