# Enhanced Malayalam IME - Scripts

This folder contains installation and setup scripts for the Enhanced Malayalam Input Method Editor.

## Scripts

- **install-ime.bat** - Main installation script
- **verify-setup.bat** - Setup verification and testing
- **check-files.bat** - File integrity checking
- **FINAL-INSTALL.bat** - Final installation steps

## Usage

Run scripts as Administrator for proper system integration.
